# weather

Simple weather app using [OpenWeatherMap](https://openweathermap.org/api).

## Getting Started

Create a `.env` file and insert your API key in `KEY=VALUE` format.

E.g. in terminal:

```sh
$ echo OPENWEATHERMAP_API_KEY=theCakeIsALie42 > .env
```
